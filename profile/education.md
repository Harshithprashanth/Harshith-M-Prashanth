### Programme Overview

The Bachelor of Engineering in Electronics and Communication Engineering provided comprehensive training in the mathematical, physical, and engineering foundations of electronic systems, signal processing, communication theory, and embedded computing. The four-year programme combined rigorous theoretical coursework with extensive laboratory work and industry exposure.

### Core Curriculum

#### Year 1 — Foundation Sciences
| Course | Description |
|--------|-------------|
| Engineering Mathematics I & II | Calculus, Linear Algebra, Differential Equations |
| Engineering Physics | Semiconductor physics, electromagnetism, optics |
| Basic Electronics | Diodes, BJTs, FETs, basic circuits |
| Introduction to Programming (C) | Fundamentals of procedural programming |
| Engineering Drawing & Design | Technical drawing and CAD fundamentals |

#### Year 2 — Core ECE Foundations
| Course | Description |
|--------|-------------|
| Network Theory & Circuits | KVL/KCL, Thevenin/Norton, transient analysis |
| Electronic Devices & Circuits | BJT/MOSFET amplifiers, biasing, frequency response |
| Signals & Systems | Fourier series/transform, Laplace transform, Z-transform |
| Digital Electronics | Boolean algebra, combinational/sequential logic, FSMs |
| Electromagnetic Theory | Maxwell's equations, wave propagation, transmission lines |
| Data Structures & Algorithms | Arrays, trees, graphs, sorting, searching |

#### Year 3 — Advanced ECE Topics
| Course | Description |
|--------|-------------|
| Digital Signal Processing | DFT/FFT, FIR/IIR design, multirate processing, adaptive filtering |
| Analog & Digital Communication | AM/FM/PM, PCM, QAM, channel coding, information theory |
| Microprocessors & Microcontrollers | 8086, ARM Cortex-M, assembly programming, peripheral interfaces |
| VLSI Design | CMOS logic, standard cell design, timing analysis, layout |
| Control Systems | Transfer functions, Bode plots, PID, state-space |
| Embedded Systems | RTOS concepts, hardware-software co-design, device drivers |

#### Year 4 — Specialisation & Project
| Course | Description |
|--------|-------------|
| Wireless Communication | OFDM, MIMO, cellular systems, 4G/5G standards |
| IoT & Sensor Networks | Sensor interfaces, MQTT, cloud integration, edge computing |
| Machine Learning (Elective) | Supervised/unsupervised learning, neural networks |
| FPGA Design & Implementation | Xilinx/Intel FPGAs, HDL programming, hardware acceleration |
| Power Electronics | Converters, inverters, motor drives |
| Final Year Project | Intelligent Fault Detection System — 94.7% classification accuracy |
